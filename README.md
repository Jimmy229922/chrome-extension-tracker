# Chrome Extension - Account Copy Tracker

## الوصف:
إضافة Chrome لتتبع الحسابات وعناوين IP والمحافظ المنسوخة

## المميزات:
- ✅ تتبع الحسابات المنسوخة (7 أرقام)
- ✅ كشف عناوين IP مع معلومات الموقع الجغرافي
- ✅ تمييز IPs من محافظات محددة (كركوك، أربيل، السليمانية، UK)
- ✅ تتبع عناوين محافظ TRON
- ✅ نظام تصنيف للحسابات (مشبوه، آمن، قيد المراجعة، محظور)
- ✅ حاسبة الأرباح للصفقات
- ✅ واجهة عربية مع دعم الوضع الليلي

## التثبيت:
1. حمّل المجلد `d:\inzo\new`
2. افتح Chrome وانتقل إلى `chrome://extensions/`
3. فعّل "وضع المطور" (Developer mode)
4. اضغط "تحميل ملحق غير معبأ" (Load unpacked)
5. اختر مجلد `d:\inzo\new`

## الاستخدام:
1. انسخ رقم حساب (7 أرقام) أو IP أو عنوان محفظة
2. الإضافة ستكتشفه تلقائياً وتحفظه
3. افتح Side Panel لرؤية السجل
4. استخدم البحث والفلاتر لتنظيم البيانات

## الملاحظات:
- البيانات مخزنة محلياً على جهازك فقط
- لا يتم مشاركة أي بيانات مع خوادم خارجية
- استخدام شخصي بالكامل

## الملفات الرئيسية:
- `manifest.json` - إعدادات الإضافة
- `background.js` - معالجة الخلفية والكشف
- `sidepanel.html/js` - الواجهة الرئيسية
- `style.css` - التنسيقات
